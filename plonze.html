Plonze

Within Project Geavanceerde Engineering we focused on parametric design and the modes of personalisation. We learned how to work with Grasshopper, within Rhino, to model compationally, which makes mass peersonalisation possible.

I made Plonze, swimming goggles fully modeled based on certain measuring points on the face of the user.

Preparation

After doing research, deciding on the modes of personalisation in the design, how to measure the necessary points and exploring manufacturing possibilities, a customer journey was made as well as a map of the personalisation and the user interactions during this process.



Coding

Now the coding can begin. Using the inputs measured previously, with a model of a french king as try-out face, the full swimming goggles are modeled. For a more detailed look at how I modeled it, you can take a look at my full design report further below.



Final model

Once fully modeled, the goggles can be exported as .stl file and sliced for their selective additive manufacturing processes. A preview can be "rendered" in Rhino itself, using custom previews for the code to show the proper colours.



Prototype

A prototype was made with the output files, which I then photographed in a lightbox for presentational uses and to add to the report. The prototype existed of a simple PLA-printed frame, and the skirts and nose bridge were printed using a flexible resin.



Report

With the photographs taken, I could build my final report in Affinity Publisher. This contains the full process, from start to finish. A simple presentation and a poster were also made.






Renders

After finishing this project, I realised I wasn't fully content with the model renders I could make in Rhino, so I decided to fully render the model in Autodesk VRED.
